This project is made with SFML3.


In this snake game you collect food to increase your snake's length. If you hit the border or yourself, you lose.

Controls:
Arrow keys for movement.



To run the game without visual studio:

1. open x64 folder
2. open release folder
3. run the Snake game application


To open the project with visual studio:

1. open Snake game.sln in the root file.
